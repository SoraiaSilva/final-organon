Beyond Sight

ITGROW | CRITICAL Software

The goal of Beyond Sight is to allow people with visual impairments to access nonverbal information. This mobile application, through the use of audio description, will communicate with its users, providing a way of understanding the contents shown in a Powerpoint, thus enabling them to autonomously attend those types of presentations without any assistance.

3 main components:

An extension for Powerpoint, able to inform the user about the content presented in the current slide;

A server that allows registration of presentations and mobile clients;

A mobile application where the information that is displayed on the current slide will be transmitted in real time.

These three parts would work together to provide visually impaired people with the tools to follow a presentation in a much less frustrating way. The application provides real-time information of changes to slides and of changes regarding the content of the slides (appearance and disappearance of text blocks, for example), and is accessible to users of screen readers, as well as compatible with desktop and mobile devices.
