HELDA - Helpful Disassembly Assistant

Peugeot CitroÃ«n Mangualde

HELDA aims to develop an application capable of helping its users in operations such as disassembly/repair in automobile workstations, alongside with the registration of occurrences for future reference and analysis.

This project takes advantage of the Android's voice recognition system and turns it into a continuous listening service, capable of identifying a set of keywords and acting according to them.

HELDA is a complete solution, that eliminates errors in the disassembly process, minimizes downtime while providing data that can be used for process refinement.
